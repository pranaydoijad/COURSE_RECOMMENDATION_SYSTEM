## 🚀 Quick Start

## 🔌 API Endpoints

### Recommendation Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/recommend` | Get recommendations for a user |
| POST | `/api/batch_recommend` | Get recommendations for multiple users |

**Parameters for GET /api/recommend:**
- `user_id` (required): User ID
- `algorithm` (optional): `hybrid`, `popularity`, `content_based` (default: `hybrid`)
- `k` (optional): Number of recommendations, 1-20 (default: 5)

| Method | Endpoint | Description |

| Method | Endpoint | Description |

| GET | `/api/courses` | Get all courses |
| GET | `/api/courses/<course_name>` | Get specific course details |

**Parameters for GET /api/courses:**
- `difficulty` (optional): `Beginner`, `Intermediate`, `Advanced`
- `sort_by` (optional): `enrollment`, `rating`, `price`, `duration`

### System Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/health` | Health check |
| GET | `/api/metrics` | Performance metrics |

## 🔐 Security & Best Practices

### For Production:

1. **Use environment variables for secrets:**
   ```python
   from dotenv import load_dotenv
   load_dotenv()
   SECRET_KEY = os.getenv('SECRET_KEY')
   ```

2. **Add API authentication:**
   ```python
   from flask_httpauth import HTTPBasicAuth
   auth = HTTPBasicAuth()
   ```

3. **Rate limiting:**
   ```python
   from flask_limiter import Limiter
   limiter = Limiter(app)
   ```

4. **HTTPS only:**
   ```python
   app.config['SESSION_COOKIE_SECURE'] = True
   ```

5. **CORS whitelist:**
   ```python
   CORS(app, resources={r"/api/*": {"origins": ["https://yourdomain.com"]}})
   ```

## 🐛 Troubleshooting

### "recommender_models.pkl not found"
**Solution:** Run the notebook first to generate the model file:
```bash
jupyter notebook Complete_Course_Recommendation_System.ipynb
```

### Streamlit stuck on "Running..."
**Solution:** Kill the process and restart
```bash
# Find the process
lsof -i :8501
# Kill it
kill -9 <PID>
# Restart
streamlit run streamlit_app.py
```

### Flask port 5000 already in use
**Solution:** Use a different port
```bash
python flask_api.py --port 5001
```

Or kill the process using port 5000:
```bash
lsof -i :5000
kill -9 <PID>
```

### Connection refused when calling API
**Solution:** Make sure both services are running
- Streamlit: http://localhost:8501
- Flask: http://localhost:5000

### Models loading slow
**Solution:** Use pickle protocol 4 for faster loading:
```python
pickle.dump(models, f, protocol=4)
```

## 📚 How It Works

### 1. **Data Flow**
```
User Input → Select Algorithm → Get Recommendations → Enrich with Details → Display
```

### 2. **Algorithm Details**

**Popularity-Based:**
- Ranks courses by enrollment numbers
- Returns top-k courses user hasn't taken
- Perfect for new users

**Content-Based:**
- Computes similarity between courses
- Uses: difficulty, price, duration, certification
- Personalizes based on user history

**Hybrid:**
- Combines popularity (60%) + content-based (40%)
- Balances accuracy and personalization
- Recommended for production

### 3. **Training Pipeline**
```
Raw Data (100K enrollments)
    ↓
EDA (Phase 1): Understand patterns
    ↓
Feature Engineering (Phase 2): Build similarity matrices
    ↓
Model Training (Phase 2): Train 3 recommenders
    ↓
Evaluation (Phase 2): Test on holdout data
    ↓
Serialization (Phase 2): Save to pickle
    ↓
Deployment (Phase 3): Load & serve
```

## 📈 Monitoring & Metrics

### Key Metrics

- **Precision@5**: % of recommendations user rated 4+ stars
- **Coverage**: % of courses recommended at least once
- **Latency**: Time to generate recommendation

### Example Monitoring Code

```python
import time
from flask import g

@app.before_request
def start_timer():
    g.start = time.time()

@app.after_request
def log_request(response):
    if hasattr(g, 'start'):
        elapsed = time.time() - g.start
        print(f"Request completed in {elapsed:.2f}s")
    return response
```

## 📝 Example Integration Code

### Python Client

```python
import requests

# Get recommendations
response = requests.get(
    'http://localhost:5000/api/recommend',
    params={
        'user_id': 123,
        'algorithm': 'hybrid',
        'k': 5
    }
)

recommendations = response.json()['recommendations']

for rec in recommendations:
    print(f"{rec['rank']}. {rec['course_name']} (${rec['price']:.2f})")
```

### JavaScript Client

```javascript
// Get recommendations
fetch('http://localhost:5000/api/recommend?user_id=123&algorithm=hybrid&k=5')
    .then(response => response.json())
    .then(data => {
        data.recommendations.forEach(rec => {
            console.log(`${rec.rank}. ${rec.course_name} ($${rec.price})`);
        });
    });
```

### cURL Examples

```bash

curl -X POST http://localhost:5000/api/batch_recommend \
```
